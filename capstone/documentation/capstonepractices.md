# Expectation: Design an Agentic System that can
- Reason across steps
- Use tools when needed
- Retrieve knowledge (RAG)
- Maintain context (memory)
- Handle ambiguity
- Operate safely

# What is Agentic Design
Thinking → Deciding → Acting → Learning → Safeguarding

# Key Capabilities

1. Reliability = TRUST , how correct is your answer
2. Safety-First = GUARDRAILS
3. Explainability
4. Evaluation Criteria
5. Assumptions
    1. User Business Persona 
    2. what is expected what is not
    3. which case will not be handled
    4. priority list 
6. Risks of Failure + Mitigation
    1. where can it fail, what is the fail safe mechanism 
7. Design Justification
    1. Why this architecture
    2. Why this model
    3. Why this tool
    4. Why this prompt
    5. Why this evaluation criteria
    6. Why this assumption
    7. Why this risk mitigation
    8. Why this explainability
